## OpenPOS Info
개발 환경: Embarcadero XE2 C++ Builder

사용한 DBMS: MySQL 5.7, (compatibled MariaDB)

호환 운영체제: Microsoft Windows XP, Vista, 7 (Embedded Edition, 64bit 운영체제 포함)
